# 贪吃蛇大作战 - Snake Battle

## 项目概述

一个基于 HTML5 Canvas 的多人贪吃蛇对战游戏，支持 2-6 名玩家同时游戏，采用暗黑风格 UI 设计。

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **图形渲染**: HTML5 Canvas API
- **音效**: Web Audio API
- **存储**: LocalStorage (保存游戏配置)

## 文件结构

| 文件 | 说明 |
|------|------|
| `index.html` | 游戏主页面，包含 UI 结构和配置模态框 |
| `css/style.css` | 暗黑风格样式表，支持响应式布局 |
| `js/main.js` | 游戏入口，初始化游戏实例 |
| `js/game.js` | 游戏主控制器，管理游戏循环和状态 |
| `js/snake.js` | 蛇类，管理单个蛇的状态和行为 |
| `js/food.js` | 食物管理器，处理食物生成和碰撞检测 |
| `js/ai.js` | AI 控制器和管理器，实现电脑玩家智能 |
| `js/ui.js` | UI 管理器，处理界面交互和配置 |
| `js/config.js` | 游戏配置管理，支持本地存储 |
| `js/audio.js` | 音效管理器，使用 Web Audio API |

## 核心功能

### 多人对战
- 支持 2-6 名玩家同时游戏
- 玩家 1-2 可选择人类玩家或电脑玩家
- 玩家 3-6 固定为电脑玩家

### 操作方式
| 按键 | 功能 |
|------|------|
| WASD | 玩家 1 移动 |
| 方向键 | 玩家 2 移动 |
| 空格键 | 暂停/继续 |
| P 键 | 切换自动/手动模式 |

### 游戏机制
- 吃到食物增加长度（食物价值 1-6，不同颜色代表不同价值）
- 撞墙或被阻挡时增加 3 格并闪烁提示
- 难度升级：每 60 秒格子数减少（20→18→16... 最小 10）

### AI 系统
- 智能寻路算法，自动寻找最近的食物
- 碰撞风险预测，避免撞墙和撞其他蛇
- 食物优先级评估，优先选择高价值食物

### 配置选项
- **玩家数量**: 2-6 人
- **初始格子数**: 10-30
- **升级间隔**: 30-180 秒
- **游戏速度**: 慢速/正常/快速/极速

## 运行方式

直接在浏览器中打开 `index.html` 文件即可运行游戏。
